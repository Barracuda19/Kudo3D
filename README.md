Kudo 3D — Printare 3D Custom

Descriere

Kudo 3D este un site-portofoliu pentru servicii de printare 3D: figurine, decor, prototipuri și piese personalizate. Proiectul oferă o pagină simplă, performantă și responsive pentru prezentarea lucrărilor, serviciilor și modalităților de contact.

Caracteristici

- Design modern, dark-theme cu glow subtil
- Galerie mixtă de imagini și clipuri video
- Lightbox pentru vizualizare imagini
- Secțiuni: Hero, Servicii, Galerie, Contact
- Responsive: grile adaptabile pentru mobil/tabletă
- Accesibilitate îmbunătățită (focus outlines, contraste verificate)

Structură fișiere

- index.html — pagina principală
- css/style.css — stiluri principale
- js/script.js — scripturi (lightbox, interacțiuni)
- images/ — imagini de portofoliu
- videos/ — clipuri video folosite în galerie
- favicon.png — pictogramă site

Instalare locală

1. Clonează sau descarcă repo în calculatorul tău.
2. Deschide `index.html` într-un browser modern (Chrome, Firefox, Edge).

Notă: proiectul nu are dependențe externe compilate; fontul `Inter` este încărcat din Google Fonts.

Dezvoltare

- Stiluri: modifică `css/style.css` pentru aspect și responsive.
- Scripturi: modifică `js/script.js` pentru comportamentul lightbox/galerie.
- Imagini și video: adaugă fișiere în folderele `images/` și `videos/` și actualizează markup-ul din `index.html`.
